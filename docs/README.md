# TVB-Child

**TVB-Child** is an open-source computational framework for modelling whole-brain dynamics in developmental neuroimaging using The Virtual Brain (TVB).

## Overview

The repository contains the custom code developed for this study, including

- whole-brain network simulations
- parameter estimation
- downstream statistical analyses
- manuscript figure generation

The repository does **not** redistribute third-party software such as

- FSL
- Human Connectome Project Pipelines
- Connectome Workbench

Instead, it contains the custom scripts that interface with these tools.

## Repository structure

```
src/        Source code
scripts/    Scripts to reproduce analyses
configs/    Configuration files

examples/   Example workflows
paper/      Material related to the manuscript
```
